# Bells University Student Recipe & Nutrition Platform

## Project Overview

This project was developed as part of the **ICT 235** course.

### Key Features

- **27 Student Recipes**
- **Nutritional Tracking**
- **Dynamic Search**
- **Interactive UI**
- **Responsive Design**

## Tech Stack & Dependencies

The project is built using the following technologies and frameworks:

- **Frontend Framework:** React.js (Vite)
- **Styling:** CSS & Bootstrap 5
- **Icons:** Bootstrap Icons
- **Carousel/Slider:** React-Slick & Slick-Carousel

## Running the App

### Prerequisites

Ensure **Node.js** is installed on your computer.

### Installation

1. **Clone the repository** (or download the source code) :
   Using the Terminal; `git clone https://github.com/Tofsonline/Student-Recipe-and-Nutrition-Platform`

2. **Navigate to the project folder**
   Using the Terminal; `cd bells-nutrition-app`

3. **Install the required dependencies**
   Using the Terminal; `npm install react-slick slick-carousel bootstrap-icons bootstrap`

4. **Run the development server**
   Using the Terminal; `npm run dev`

5. **View the app**
   Open your browser and go to `http://localhost:5173`.

**Or Using Github Pages:**
https://tofsonline.github.io/Student-Recipe-and-Nutrition-Platform/
